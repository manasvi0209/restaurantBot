# Restaurant Ordering Chatbot

This project is a simple restaurant chatbot built using **Python, FastAPI, and Gradio**.
It allows users to view the menu, order food items, cancel items, and finalize orders through a chat interface.

## Features

* View menu
* Add items to order
* Cancel items from order
* View current order
* Finalize order (takeaway or delivery)

## Technologies Used

* Python
* FastAPI
* Gradio
* Uvicorn

## How to Run

Install dependencies:

pip install fastapi uvicorn gradio requests pydantic

Run the program:

python main.py

Then open the Gradio interface in your browser.

## Example Commands

* menu
* 2 pizza
* cancel burger
* order
* finalize order
